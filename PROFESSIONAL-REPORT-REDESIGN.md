- 1180px readable desktop content width
- responsive mobile and iPad layouts
- dark evidence-report hero with readable white text
- compact cards with clear visual hierarchy
- line, bar, donut, gauge and radar visualisations
- tabs and collapsible/detail-first presentation instead of long unstructured pages

## Unified Google report

`/reports/organic-search/` and `/reports/traffic-analytics/` now use the same report interface.

A connected user can switch inside the report between:

1. Google Search Console
2. Google Analytics 4

The report provides:

- matching property recommendation
- 7/28/90/180/365-day and custom ranges
- previous equivalent-period comparison
- source-specific metric cards and charts
- GSC tabs: Queries, Pages, Countries, Devices, Search appearance, Sitemap & URLs
- GA4 tabs: Channels, Landing pages, Source/medium, Countries, Cities, Devices
- rows-per-page options: 25, 50 and 100
- Previous, Page X of Y and Next pagination
- CSV export and Print/PDF controls

## Backlink resilience retained

Gemini assistance remains optional. When the Gemini key is unavailable, quota is exhausted, the model fails, or the request times out, backlink discovery continues through deterministic public-search discovery and live HTML link verification.
