# Tool Matrix

| Tool | Primary source | Output |
|---|---|---|
| Complete SEO Growth Report | Live crawl, PageSpeed, public signals, AI synthesis | Executive report, health categories, CWV, content/schema/links, roadmap |
| Keyword Intelligence | Gemini analysis | Intent, clusters, content plan and clearly labelled AI demand ranges |
| PageSpeed and Core Web Vitals | PageSpeed Insights and CrUX | Mobile/desktop gauges, lab metrics, field CWV, exact resource opportunities |
| Whole Website SEO Audit | Live site crawler, RDAP, optional OpenPageRank | Technical, on-page, content, schema, link and authority findings |
| Organic Search Performance | Google Search Console | Clicks, impressions, CTR, position, trends, opportunities and breakdowns |
| Traffic and Analytics | Google Analytics 4 | Users, sessions, engagement, channels, landing pages, geography and devices |
| Schema Intelligence | Live crawl plus deterministic generator and AI recommendations | Existing schema analysis, errors, recommended types and valid JSON-LD generation |
| AI Execution Roadmap | Completed evidence plus business inputs | Prioritised 30/60/90-day plan without invented metrics |

Verified data, live crawl evidence, public metrics and AI recommendations are visually labelled separately.
