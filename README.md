# Pajee SEO V10 — Replacement Website

This is the complete GitHub/Vercel root package for the structurally redesigned Pajee SEO website.

## Included

- Modern, service-specific animated homepage cards.
- Rebuilt Services Hub and conversion-focused individual service pages.
- Responsive desktop hover mega menu and mobile accordion menu.
- Complete SEO Growth, PageSpeed, Website Audit, Organic Visibility, Traffic, Roadmap and Schema reporting interfaces.
- Backlink & Authority Intelligence tool with public candidate discovery, live-link verification and Search Console Links CSV import.
- Strict Gemini model configuration using `gemini-3.5-flash` only.

## Deploy

Upload the **contents of this folder** directly to the repository root. The repository root must show:

```text
index.html
vercel.json
package.json
api/
lib/
assets/
services/
tools/
results/
reports/
```

Read `DEPLOY-V10.md` and `API-KEY-SETUP.md` before deployment.

## Accuracy policy

- Public and AI-assisted estimates are labelled separately from verified GSC/GA4 data.
- INP is shown only from real CrUX field data.
- The backlink report is a verified public sample or an imported evidence set; it does not claim access to Google's complete backlink index.
- The Pajee Authority Signal is a Pajee SEO analytical score, not Google PageRank or Ahrefs DR.
